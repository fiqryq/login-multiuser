Hello, temans!

Sistem login ini dibuat dengan codeigniter versi 3.x dan mysql untuk databasenya. Konsepnya sederhana. Memanfaatkan session untuk menyimpan data termasuk jenis usernya saat proses login. Setelah didapatkan jenis usernya, arahkan ke controller masing-masing jenis user.
